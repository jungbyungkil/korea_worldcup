# Frontend (Vite + React)

## Windows에서 `node` / `npm` 오류가 나면

👉 [`../docs/WINDOWS-NODE-NPM.md`](../docs/WINDOWS-NODE-NPM.md)

## 실행

```powershell
cd frontend
npm install
npm run dev
```

- API 주소: `frontend/.env` 의 `VITE_API_BASE_URL` (기본 `http://localhost:8000`)
